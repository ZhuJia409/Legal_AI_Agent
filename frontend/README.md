# 前端目录

用于存放 Next.js 前端应用相关文件，包括页面、组件、样式、状态管理、AI 聊天界面、文档上传界面、引用来源展示和前端配置文件。

## 已配置内容

- `package.json`：Next.js、React、TypeScript、Tailwind CSS、shadcn/ui 兼容依赖和 AI SDK 依赖。
- `src/app/`：App Router 页面、布局和全局样式。
- `components.json`：shadcn/ui 配置。
- `eslint.config.mjs`、`tsconfig.json`：代码检查和类型检查配置。

## 常用命令

```powershell
pnpm.cmd install
pnpm.cmd lint
pnpm.cmd typecheck
pnpm.cmd dev
```
