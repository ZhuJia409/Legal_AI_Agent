# 06. 融资租赁合同规则包

- 法律依据：《民法典》第735-760条。
- 适用场景：出租人根据承租人选择购买租赁物并出租给承租人使用，承租人支付租金的融资性交易。
- 路由识别特征：出现“融资租赁、出卖人、出租人、承租人、租赁物购买、租金构成、留购价、所有权归属”等表达。
- 必查条款：三方关系、租赁物选择、购买合同衔接、交付验收、租金构成、所有权归属、留购安排、维修保险、索赔权转让、违约取回。
- 专项审查重点：检查是否构成名为租赁实为借贷，租赁物是否真实存在，承租人选择权和索赔权是否明确，租金和残值安排是否可执行，取回租赁物规则是否合规。
- 特殊陷阱：售后回租无真实租赁物；租赁物权属登记缺失；租金表不透明；承租人承担出卖人全部违约风险但无索赔权；留购价与所有权转移条件冲突。
- 输出给主智能体的风险项：名为租赁实为借贷风险、租赁物真实性风险、权属登记风险、租金结构不透明风险、索赔权缺失风险、取回规则争议风险。
