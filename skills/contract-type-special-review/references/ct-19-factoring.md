# 19. 保理合同规则包

- 法律依据：《民法典》第761-769条。
- 适用场景：应收账款债权人将现有或将有应收账款转让给保理人，保理人提供融资、管理、催收或担保付款服务。
- 路由识别特征：出现“保理、应收账款、债权转让、回购、明保理、暗保理、融资、买方付款”等表达。
- 必查条款：应收账款真实性、转让范围、通知债务人、融资额度、回购条件、追索权、有无追索、基础合同、重复转让、登记。
- 专项审查重点：检查应收账款是否真实合法可转让，通知和登记安排，基础合同抗辩，追索和回购责任，未来应收账款识别。
- 特殊陷阱：虚假应收账款；同一债权重复保理；禁止转让条款影响；暗保理通知缺失；回购责任实质构成借贷担保。
- 输出给主智能体的风险项：应收账款真实性风险、重复转让风险、通知登记风险、基础合同抗辩风险、回购追索风险、禁止转让风险。
