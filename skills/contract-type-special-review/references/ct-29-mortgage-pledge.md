# 29. 抵押/质押合同规则包

- 法律依据：《民法典》物权编担保物权规则；《担保制度司法解释》。
- 适用场景：以不动产、动产、权利、股权、应收账款等设定抵押或质押担保。
- 路由识别特征：出现“抵押、质押、担保物、登记、交付、优先受偿、担保范围、实现担保物权”等表达。
- 必查条款：主债权、担保物描述、担保范围、登记或交付、权属和负担、保险保管、处分限制、实现方式、担保期间、反担保。
- 专项审查重点：区分抵押和质押的设立要件；检查登记、交付、权属、优先顺位；检查流押流质禁止和担保范围。
- 特殊陷阱：未办理登记或交付导致担保未设立/不得对抗；担保物描述不特定；流押流质条款；重复抵质押；公司担保决议缺失。
- 输出给主智能体的风险项：担保设立风险、登记交付风险、担保物特定化风险、优先顺位风险、流押流质风险、公司担保授权风险。
