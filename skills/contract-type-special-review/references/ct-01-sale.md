# 01. 买卖合同规则包

- 法律依据：《民法典》第595-647条。
- 适用场景：一方转移标的物所有权，另一方支付价款；包括一般买卖、分期付款买卖、样品买卖、试用买卖、招投标买卖、所有权保留买卖等。
- 路由识别特征：出现“买卖、采购、销售、货物、交付、验收、质量标准、价款、所有权转移、风险转移”等表达。
- 必查条款：标的物特定化、数量质量、价款及付款节点、交付地点和期限、验收标准、质量异议期、所有权和风险转移、违约责任、发票税费、售后和质保。
- 专项审查重点：检查标的是否明确，检验期限是否合理，分期付款解除条件是否合规，样品与质量标准是否绑定，招投标合同是否背离中标文件，所有权保留是否约定清楚。
- 特殊陷阱：所有权转移与风险转移混淆；验收合格被默认为放弃质量异议；质量标准只写“合格”但无可执行指标；中标后合同实质性背离招标文件；卖方保留所有权但未约定取回和处分规则。
- 输出给主智能体的风险项：标的/质量不明确风险、验收与异议期风险、付款与交付不同步风险、所有权/风险转移冲突风险、招投标实质背离风险、质保责任不足风险。
