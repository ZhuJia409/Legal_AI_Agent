# 17. 合伙合同规则包

- 法律依据：《民法典》第967-978条。
- 适用场景：两个以上合伙人为共同事业共享利益、共担风险，不一定设立合伙企业。
- 路由识别特征：出现“合伙、共同经营、出资、利润分配、亏损承担、合伙事务、退伙、清算”等表达。
- 必查条款：出资方式、出资期限、事务执行、表决机制、利润分配、亏损承担、竞业限制、入伙退伙、解散清算、债务承担。
- 专项审查重点：检查合伙事务执行权，合伙债务承担，利润亏损分配是否一致，退伙和清算机制是否可执行。
- 特殊陷阱：只约定收益不约定亏损；个人合伙与公司合资混淆；对外债务承担未揭示；退伙估值缺失；僵局处理缺失。
