# 14. 物业服务合同规则包

- 法律依据：《民法典》第937-950条；《物业管理条例》。
- 适用场景：物业服务人向业主、业主大会或建设单位提供物业管理服务。
- 路由识别特征：出现“物业服务、业主、物业费、公共区域、维修养护、前期物业、业主大会”等表达。
- 必查条款：服务范围、服务标准、物业费、公共收益、维修养护、安全管理、业主配合、合同期限、退出交接、违约责任。
- 专项审查重点：检查服务标准是否量化，公共收益归属，前期物业合法性，费用调整程序，退出交接和资料移交。
- 特殊陷阱：物业费调整无程序；公共收益被物业服务方单方占有；安全保障义务无限扩大；前期物业与业主大会选聘冲突；退出交接缺失。
- 输出给主智能体的风险项：服务标准不明确风险、费用调整风险、公共收益风险、安全责任风险、退出交接风险、前期物业效力风险。
