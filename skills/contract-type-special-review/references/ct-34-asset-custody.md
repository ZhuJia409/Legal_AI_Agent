# 34. 资产托管合同规则包

- 法律依据：《证券投资基金法》；《信托法》；《企业年金基金管理办法》；《证券投资基金托管业务管理办法》。
- 适用场景：基金托管、信托财产托管、企业年金托管、资管产品托管等金融资产托管安排。
- 路由识别特征：出现“资产托管、托管人、管理人、托管账户、资产隔离、估值核算、投资监督、划款指令”等表达。
- 必查条款：托管人资质、托管资产范围、账户开立、资产隔离、估值核算、投资监督、指令审核、费用、信息披露、责任限制、终止交接。
- 专项审查重点：检查托管人与管理人职责边界，资产独立性和账户隔离，投资指令审核，估值核对，托管人责任标准和免责范围。
- 特殊陷阱：把资产托管误写成普通保管；托管人责任被过度免除；资产隔离条款不足；投资监督流于形式；划款指令审核标准不清。
- 输出给主智能体的风险项：托管资质风险、资产隔离风险、职责边界风险、估值核算风险、投资监督风险、终止交接风险。
