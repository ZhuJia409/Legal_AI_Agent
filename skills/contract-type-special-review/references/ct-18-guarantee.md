# 18. 保证合同规则包

- 法律依据：《民法典》第681-702条；《担保制度司法解释》。
- 适用场景：保证人为债务人的债务履行向债权人提供保证。
- 路由识别特征：出现“保证、担保、保证人、主债权、一般保证、连带责任保证、保证期间”等表达。
- 必查条款：主债权、保证方式、保证范围、保证期间、保证责任触发、最高额保证、追偿权、反担保、独立性表述。
- 专项审查重点：检查保证方式是否明确，不明确时的法律后果；保证期间是否有效；最高额保证是否有额度和期间；主合同变更对保证责任的影响。
- 特殊陷阱：保证期间写成“至债务清偿完毕”导致争议；一般保证与连带保证混淆；保证范围过宽；未经保证人同意变更主债务；公司对外担保决议缺失。
- 输出给主智能体的风险项：保证方式不明风险、保证期间风险、保证范围过宽风险、公司担保授权风险、主合同变更风险、追偿安排缺失风险。
