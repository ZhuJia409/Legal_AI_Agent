# 30. 知识产权许可合同规则包

- 法律依据：《专利法》；《商标法》；《著作权法》；《民法典》技术合同章节；《技术进出口管理条例》。
- 适用场景：专利、商标、著作权、软件、技术秘密等知识产权许可。
- 路由识别特征：出现“许可、专利、商标、著作权、软件许可、地域、期限、独占、排他、普通许可、改进成果”等表达。
- 必查条款：许可权利、许可类型、地域期限、用途范围、再许可、许可费、权利维持、侵权处理、质量控制、改进成果、终止后处理。
- 专项审查重点：检查许可人权利基础，许可类型和范围，商标质量控制，专利有效性，软件复制和部署限制，技术进出口限制。
- 特殊陷阱：独占/排他/普通许可混淆；许可范围小于业务用途；未约定再许可；权利失效责任缺失；改进成果归属不清。
- 输出给主智能体的风险项：权利基础风险、许可范围不足风险、再许可风险、权利维持风险、侵权处理风险、改进成果风险。
