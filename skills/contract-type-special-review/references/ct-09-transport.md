# 09. 运输合同规则包

- 法律依据：《民法典》第809-842条。
- 适用场景：货运、客运、多式联运、物流配送、仓配一体等。
- 路由识别特征：出现“运输、承运人、托运人、收货人、运费、提单、装卸、货损、延误、路线”等表达。
- 必查条款：运输标的、路线和方式、交付时间、运费、包装装卸、风险转移、货损货差、延误责任、保险、单证、索赔期限。
- 专项审查重点：检查承运人资质，货物信息和包装责任，交接节点和风险转移，赔偿限额是否合理，延误和不可抗力处理。
- 特殊陷阱：货损赔偿上限过低；交接节点不清导致责任断档；危险品申报缺失；多式联运责任主体不明；签收即视为无质量问题。
- 输出给主智能体的风险项：货损货差风险、延误责任风险、危险品合规风险、赔偿上限风险、交接证据风险、多式联运责任风险。
