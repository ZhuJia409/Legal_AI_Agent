# 08. 建设工程合同规则包

- 法律依据：《民法典》第788-808条；《建筑法》；《建设工程司法解释》。
- 适用场景：工程勘察、设计、施工、总承包、分包、监理、工程价款结算等。
- 路由识别特征：出现“工程、施工、发包、承包、总包、分包、竣工验收、工程款、签证、索赔、质保金”等表达。
- 必查条款：资质许可、工程范围、工期、质量标准、价款计价、变更签证、竣工验收、结算审计、质保期和质保金、安全责任、分包管理。
- 专项审查重点：检查承包资质和发包程序，违法分包转包，黑白合同，工程价款和结算程序，竣工验收与付款联动，优先受偿权和质保安排。
- 特殊陷阱：无资质或借用资质；违法分包/转包；阴阳合同；以审计作为无限期付款前提；签证授权不清；质保期和质保金混淆。
- 输出给主智能体的风险项：合同效力风险、资质缺失风险、违法分包风险、工程款结算风险、竣工验收风险、质保金返还风险、安全责任风险。
