# 04. 借款合同规则包

- 法律依据：《民法典》第667-680条；《民间借贷司法解释》。
- 适用场景：金融借款、民间借贷、企业间借款、循环借款、展期或重组借款等。
- 路由识别特征：出现“借款、本金、利息、还款、借款期限、贷款、授信、展期、逾期利率、担保”等表达。
- 必查条款：借款金额、用途、期限、利率、还款计划、提款条件、提前还款、逾期责任、担保、违约事件、加速到期、费用和税费。
- 专项审查重点：检查利率和综合成本是否合法，借款用途是否被限制，提前到期触发条件是否合理，担保与主债权是否对应，展期是否形成新债或原债延续。
- 特殊陷阱：利息、服务费、咨询费叠加规避利率上限；空白借据或倒签；借款用途违法；加速到期条件过宽；担保期间与债务期限不匹配。
- 输出给主智能体的风险项：利率违法风险、还款安排不明确风险、用途违规风险、加速到期失衡风险、担保不匹配风险、展期效力风险。
