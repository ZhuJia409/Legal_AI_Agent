# 21. 保密协议 / NDA 规则包

- 法律依据：《民法典》合同编；《反不正当竞争法》商业秘密规则；相关知识产权法律。
- 适用场景：披露方与接收方围绕商业秘密、技术信息、经营信息等设定保密义务。
- 路由识别特征：出现“NDA、保密协议、保密信息、商业秘密、披露方、接收方、允许披露、保密期限”等表达。
- 必查条款：保密信息定义、例外信息、使用目的、接触范围、保护措施、返还销毁、保密期限、违约责任、禁令救济、管辖。
- 专项审查重点：检查保密信息范围是否过宽或过窄，例外是否合理，保密期限是否覆盖商业秘密存续，违约赔偿和禁令救济是否可执行。
- 特殊陷阱：把所有信息都定义为保密信息导致执行困难；保密期限过短；未约定返还销毁；允许披露对象过宽；违约金过低无法覆盖泄密损失。
- 输出给主智能体的风险项：保密范围风险、例外条款风险、期限不足风险、返还销毁缺失风险、违约责任不足风险、允许披露失控风险。
