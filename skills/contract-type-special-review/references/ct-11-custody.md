# 11. 保管合同规则包

- 法律依据：《民法典》第888-903条。
- 适用场景：保管人保管寄存人交付的物品并返还，包括有偿或无偿保管。
- 路由识别特征：出现“保管、寄存、保管物、保管费、领取、返还、毁损灭失”等表达。
- 必查条款：保管物描述、交付和领取、保管期间、保管费、保管标准、禁止使用、毁损灭失责任、孳息归属、留置权。
- 专项审查重点：检查保管物状态和价值是否记录，保管人注意义务标准，领取凭证和返还条件，贵重物品声明规则。
- 特殊陷阱：保管物价值未声明导致赔偿争议；无偿保管责任标准被误用；保管人擅自使用保管物；领取凭证缺失；与资产托管混淆。
- 输出给主智能体的风险项：保管物识别风险、赔偿范围风险、返还证据风险、擅自使用风险、留置权风险、贵重物品声明风险。
