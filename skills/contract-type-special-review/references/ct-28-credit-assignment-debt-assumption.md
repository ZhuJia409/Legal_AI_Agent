# 28. 债权转让/债务承担协议规则包

- 法律依据：《民法典》第545-556条。
- 适用场景：债权人转让债权，债务转移或第三人加入债务。
- 路由识别特征：出现“债权转让、债务承担、债务转移、债权通知、债务加入、受让方、原债权人”等表达。
- 必查条款：基础债权债务、转让范围、通知债务人、从权利、抗辩和抵销、债务承担同意、担保延续、价款、交割文件。
- 专项审查重点：检查债权是否可转让，通知是否生效，债务承担是否经债权人同意，从权利和担保是否随同转移，抗辩权是否保留。
- 特殊陷阱：未通知债务人导致不能对抗；禁止转让条款处理不当；债务承担未经债权人同意；担保人未同意导致担保脱保；债权真实性瑕疵。
- 输出给主智能体的风险项：通知生效风险、禁止转让风险、债权真实性风险、债务承担同意风险、担保脱保风险、抗辩抵销风险。
