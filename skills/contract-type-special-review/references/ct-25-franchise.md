# 25. 特许经营合同规则包

- 法律依据：《商业特许经营管理条例》；《民法典》。
- 适用场景：特许人授权被特许人使用商标、商号、经营模式、技术和运营体系开展经营。
- 路由识别特征：出现“特许经营、加盟、品牌授权、加盟费、持续经营指导、统一运营、区域保护”等表达。
- 必查条款：特许人资质和备案、授权范围、品牌和知识产权、加盟费、运营标准、培训支持、区域保护、采购限制、信息披露、解除退出。
- 专项审查重点：检查“两店一年”和备案信息，信息披露义务，授权范围和期限，经营标准和检查权，加盟费退还，商标许可和区域限制。
- 特殊陷阱：特许人未备案或资质不足；加盟费不退但特许人支持缺失；区域保护空泛；强制采购价格不透明；品牌授权与商标权不匹配。
- 输出给主智能体的风险项：备案资质风险、信息披露风险、授权范围风险、加盟费返还风险、区域保护风险、强制采购风险。
