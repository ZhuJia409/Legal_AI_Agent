# 02. 供用电/水/气/热力合同规则包

- 法律依据：《民法典》第648-656条。
- 适用场景：供电、供水、供气、供热等连续供应合同。
- 路由识别特征：出现“供电、供水、供气、供热、用量、计量、管网、停供、抢修、阶梯价格、抄表”等表达。
- 必查条款：供应主体资质、供应范围、计量方式、计费标准、费用结算、供应质量、停供条件、抢修义务、安全责任、违约责任。
- 专项审查重点：检查供应主体是否具备资格，计量和结算是否可核验，停供是否设置通知和例外，供应质量及抢修责任是否明确，价格调整机制是否透明。
- 特殊陷阱：单方停供权过宽；价格调整未设置通知和确认；计量设备归属和误差处理缺失；不可抗力与计划检修边界不清；用户安全责任被无限扩大。
- 输出给主智能体的风险项：停供条件不合理风险、计量结算争议风险、价格调整不透明风险、供应质量责任缺失风险、安全责任分配失衡风险。
