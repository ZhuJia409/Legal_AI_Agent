# 20. 劳动合同规则包

- 法律依据：《劳动合同法》；《劳动法》；《社会保险法》。
- 适用场景：用人单位与劳动者建立劳动关系，包括固定期限、无固定期限、试用期、竞业限制等。
- 路由识别特征：出现“劳动合同、岗位、工资、工时、试用期、社保、竞业限制、解除、员工”等表达。
- 必查条款：合同期限、工作内容和地点、工时休假、劳动报酬、社保福利、试用期、保密和竞业限制、培训服务期、解除终止、经济补偿。
- 专项审查重点：检查必备条款是否完整，试用期是否合法，竞业限制补偿是否约定，解除条件是否符合法定限制，社保缴纳是否合规。
- 特殊陷阱：用劳务协议规避劳动关系；试用期重复或超期；竞业限制无补偿；违法约定员工承担违约金；解除条件突破强制规定。
- 输出给主智能体的风险项：劳动关系规避风险、试用期违法风险、薪酬社保风险、竞业限制无效风险、违法解除风险、违约金条款风险。
