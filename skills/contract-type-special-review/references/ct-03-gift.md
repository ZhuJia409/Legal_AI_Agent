# 03. 赠与合同规则包

- 法律依据：《民法典》第657-666条。
- 适用场景：一方无偿向另一方转移财产权益，包括一般赠与、附义务赠与、公益赠与、经过公证的赠与。
- 路由识别特征：出现“赠与、无偿、捐赠、附义务、撤销、公益、交付”等表达。
- 必查条款：赠与标的、交付方式、权利瑕疵、附随义务、撤销权、税费承担、公益目的、违约和返还机制。
- 专项审查重点：检查是否属于可任意撤销赠与，附义务是否明确，公益或公证赠与是否排除任意撤销，赠与财产权属和负担是否清晰。
- 特殊陷阱：以赠与名义隐藏买卖、代持或利益输送；附义务表述不清导致履行争议；赠与人撤销权未被充分识别；税费承担缺失。
- 输出给主智能体的风险项：赠与可撤销风险、附义务不明确风险、权属瑕疵风险、名为赠与实为交易风险、税费争议风险。
