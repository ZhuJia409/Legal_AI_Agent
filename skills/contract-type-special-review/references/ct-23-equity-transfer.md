# 23. 股权转让协议规则包

- 法律依据：《公司法》；《民法典》。
- 适用场景：有限责任公司或股份公司股权、股份、出资份额转让。
- 路由识别特征：出现“股权转让、转让方、受让方、目标公司、交割、工商变更、优先购买权、出资”等表达。
- 必查条款：标的股权、转让价款、付款安排、交割条件、股东同意、优先购买权、工商变更、陈述保证、债务披露、违约责任。
- 专项审查重点：检查转让方是否有权处分，其他股东优先购买权，未实缴出资责任，目标公司债务和或有负债，交割和变更登记安排。
- 特殊陷阱：股权代持未披露；未实缴出资责任转移不清；优先购买权程序瑕疵；价款支付早于交割保护不足；目标公司隐性债务。
- 输出给主智能体的风险项：处分权风险、优先购买权风险、未实缴出资风险、交割登记风险、或有负债风险、陈述保证不足风险。
