# 05. 租赁合同规则包

- 法律依据：《民法典》第703-734条。
- 适用场景：房屋、设备、车辆、场地等租赁关系。
- 路由识别特征：出现“租赁、租金、租期、租赁物、押金、转租、续租、维修、返还、交付使用”等表达。
- 必查条款：租赁物、租期、租金和押金、交付状态、用途限制、维修责任、转租规则、续租权、提前解除、返还验收、违约责任。
- 专项审查重点：检查租赁期限是否超过20年，出租人权属和处分权是否明确，转租和装修改造是否受控，维修责任和费用分担是否清楚，押金返还条件是否合理。
- 特殊陷阱：租期超过法定上限；“买卖不破租赁”例外未识别；抵押在先租赁在后导致不能对抗抵押权；押金扣除条件过宽；装修归属和补偿缺失。
- 输出给主智能体的风险项：租期效力风险、权属瑕疵风险、转租违规风险、维修责任争议风险、押金返还风险、租赁物返还风险。
