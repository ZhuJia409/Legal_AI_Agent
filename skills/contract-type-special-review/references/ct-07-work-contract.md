# 07. 承揽合同规则包

- 法律依据：《民法典》第770-787条。
- 适用场景：加工、定作、修理、复制、测试、检验等完成工作成果并交付的合同。
- 路由识别特征：出现“加工、定作、承揽、工作成果、材料提供、图纸、验收、返工、修理”等表达。
- 必查条款：工作成果标准、材料提供、图纸和技术要求、进度节点、验收方式、返工责任、知识产权、保密、留置权、解除权。
- 专项审查重点：检查成果标准是否可验收，定作人协助义务是否明确，材料瑕疵责任是否分配，承揽人是否可转包，任意解除权及损失补偿是否处理。
- 特殊陷阱：成果标准抽象导致无法验收；承揽和劳动关系混淆；未经同意转包；材料由谁提供不明；定作人任意解除但补偿缺失。
- 输出给主智能体的风险项：成果验收不明确风险、材料责任争议风险、转包风险、知识产权归属风险、任意解除补偿风险、留置权风险。
