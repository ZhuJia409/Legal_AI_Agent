# 16. 中介合同规则包

- 法律依据：《民法典》第961-966条。
- 适用场景：中介人向委托人报告订约机会或提供媒介服务，委托人支付报酬。
- 路由识别特征：出现“中介、居间、媒介、撮合、订约机会、佣金、服务费、独家委托”等表达。
- 必查条款：中介事项、服务范围、成交标准、佣金计算、支付条件、独家期、跳单责任、信息披露、费用承担。
- 专项审查重点：检查佣金触发条件是否明确，中介是否履行如实报告义务，跳单条款是否合理，独家委托期限和范围是否过宽。
- 特殊陷阱：未成交也收取高额佣金；跳单条款过宽；中介隐瞒重大事实；双边收费利益冲突；服务范围和成交因果关系不清。
- 输出给主智能体的风险项：佣金支付风险、跳单责任风险、信息披露风险、独家委托风险、利益冲突风险、费用承担风险。
