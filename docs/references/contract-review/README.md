# 合同审查模块设计文档

本目录存放「合同审查」模块的方案设计文档（HTML 格式，可直接用浏览器打开）。

源文件位于 `D:\Project\ContractModuleDesign\solution\`，此处为参考副本，原文件保留不动。

## 版本关系

| 文件                     | 说明 |
|------------------------|------|
| `01.合同审查完整方案_合并版.html` | **原方案**：合同审查完整方案（流程手册 × 类型知识库），单流程合并版。 |
| `02.合同审查_并联DAG版.html`  | **并联 DAG 拆分版**：在 01 的基础上，把审查流程拆分为多个**并联**执行的子模块（DAG 编排），是本目录当前的演进方向。 |

> 注：源目录中还存在 `02`（企业级智能体系统设计）、`03`（端到端 Harness 设计方案）两个中间探索版本，本次未纳入本目录；如需补全版本脉络，可将它们一并复制进来。

## 如何使用

- 用浏览器直接打开对应 `.html` 文件即可查看完整方案。
- 文件命名保留 `01`/`04` 数字前缀，以体现版本演进顺序，请勿去除编号重命名。
- 后续若有新版本（如 `05`），按相同规则命名后放入本目录，并在本表追加一行说明。
